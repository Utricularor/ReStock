# ReStock
ReStockは買い物の際に家に何があるのかを一目でわかるようにし、重複した買い物をなくすためのアプリです。

「帰り道に買い物に行こうと思ったけど、家に何があったか忘れた…」なんて経験はありませんか？
そういった場合でも ReStock があればその問題を解決できます。
